# Changelog

All notable changes to this project will be documented in this file.

## [1.3.0] - 2026-02-01

### Added

- Initial professional release of the Portfolio Content Management System.
- Comprehensive documentation suite including LICENSE, CONTRIBUTING.md, and CODE_OF_CONDUCT.md.
- GitHub collaboration templates (Issue & Pull Request).
- Refined README with setup guides, features, and technical details.
- Secure Admin Dashboard powered by Laravel Breeze and Tailwind CSS.
- Dynamic content management for Hero, About, Services, and Categories.
- Advanced data table interactions with Yajra Datatables.
- Excel export functionality for portfolio data analysis.
- Real-time action feedback via Toast notifications.
- Improved .gitignore for clean repository maintenance.
